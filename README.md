Ocean View Resort Web Application

Ocean View Resort Web Application is a Java EE web-based system developed to manage resort reservations and guest services.
The application helps staff handle guest registration, room reservations, and billing operations efficiently while preventing booking conflicts.

This system simplifies manual resort management by providing an organized interface for managing guests, reservations, and payments.

Technologies Used:
- Java (JDK 17)
- Java EE (JSP & Servlets)
- Apache Tomcat
- MySQL
- JDBC
- HTML
- CSS
- JavaScript
- Git & GitHub

Features:
- User login system
- Guest registration
- Room reservation management
- Room availability checking
- Prevention of overlapping bookings
- Guest billing generation
- Bill summary view
- Simple report generation

System Modules:
Authentication Module
Handles user login and access to the system.

Guest Management
Allows staff to register and manage guest information.

Room Reservation
Manages room bookings and ensures that rooms are not double-booked.

Billing System
Generates bills for guests based on their room reservations.

Reservation Records
Stores reservation history and booking details.

How to Run the Project
1. Clone the Repository
   git clone https://github.com/yasindupawan/ocean_view_resort_webapp.git

2. Open the Project

Import the project into one of the following IDEs:
IntelliJ IDEA
Eclipse
NetBeans   

3. Configure MySQL Database
Install MySQL
Create a database:
CREATE DATABASE ocean_view_resort;
Import the SQL file if available.

4. Configure Database Connection
Update your database credentials in the project configuration.

5. Deploy the Project
Install Apache Tomcat
Add the project to the Tomcat server
Start the server

Run the Application
Open your browser and navigate to:
http://localhost:8080/ocean_view_resort_webapp
